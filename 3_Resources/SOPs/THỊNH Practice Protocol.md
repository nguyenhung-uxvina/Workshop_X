# 🌱 THỊNH Practice Protocol — Hệ Thống Vận Hành Galaxy

> *Nguồn: IPARAG_AI_SecondBrain_vB_Synthesis.md — ARCHITECT Framework (Phần 6)*

## 5 Elements của THỊNH

| Chữ | Element | Hành động cốt lõi | Anti-pattern |
|-----|---------|-------------------|--------------|
| **T** | **T**hu — Capture | Ghi nhanh + ghi TẠI SAO | Capture không context |
| **H** | **H**óa — Atomize | 1 note = 1 concept, lời của mình | Copy-paste từ nguồn |
| **I** | **I**ch — Connect | Mỗi note mới: ≥ 2 wikilinks | Orphan notes |
| **N** | **N**hớ — Persist | CLAUDE.md: update hàng tháng | Context drift |
| **H** | **H**ành — Harvest | Dùng Galaxy cho decisions | Vault = graveyard |

---

## THỊNH Daily Check (2 phút cuối ngày)

```
1. T: Tuần này tôi capture bao nhiêu với context "tại sao"?
2. H: Ghi chú cuối tôi tạo có thực sự atomic không — hay chứa 2+ concepts?
3. I: Ghi chú cuối tôi có ≥ 2 meaningful links không?
4. N: CLAUDE.md có phản ánh đúng current state của tôi không?
5. H: Tuần này tôi có dùng Galaxy để inform ≥ 1 decision chưa?
```

**Healthy system = 5/5 "Có" trong 10 ngày liên tiếp.**

## 6 Drills Định Kỳ

### Drill 1 — THỊNH Daily Check
- **Tần suất:** 2 phút cuối mỗi ngày làm việc với vault
- **Mục tiêu:** Early warning khi element nào đang slip

### Drill 2 — Manual Sprint *(cho AI Dependency Counter)*
- **Tần suất:** 1 lần/tháng (thay thế 1 AI session)
- **Protocol:** Làm TẤT CẢ vault tasks thủ công (triage Inbox, tạo permanent notes, tìm connections). Journal sau: điều gì khó? Điều gì dễ hơn nghĩ?
- **Mục tiêu:** Activate B4 (Judgment Recovery); calibrate actual dependency level
- **Thành công:** Sau 3 tháng, manual triage time < 50% so với tháng đầu

### Drill 3 — Context Drift Test *(1 lần/tuần, 5 phút)*
- **Protocol:** Viết 3 predictions về Claude output TRƯỚC khi mở session. So sánh với actual. Log divergences.
- **Thành công:** Prediction accuracy > 70% → context healthy; < 50% → update CLAUDE.md ngay

### Drill 4 — Link Traversal Walk *(1 lần/tuần, 15 phút)*
- **Protocol:** Chọn 1 note ngẫu nhiên trong Galaxy. Follow links tự nhiên 15 phút, không search.
- **Thành công:** Tìm được ≥ 1 connection không nhớ mình đã biết

### Drill 5 — Harvest or Archive Audit *(1 lần/tháng)*
- **Protocol:** Review Projects + Areas. Với mỗi active project: "Tôi đã extract ≥ 1 permanent note từ project này chưa?" Nếu chưa → extract hoặc archive.
- **Thành công:** 100% active projects có ≥ 1 associated Galaxy note

### Drill 6 — Galaxy Quality Audit *(1 lần/quý)*
- **Protocol:** Chọn ngẫu nhiên 20 notes. Đánh giá: (a) Still accurate? (b) Still relevant? (c) Still atomic? Notes fail 2/3 → update hoặc archive.
- **Thành công:** < 10% stale/inaccurate

## 90-Phút Deep Work Session

```
00-10 min  THỊNH Check — identify focus của session
10-30 min  THU — Inbox processing (≥ 5 items, ≥ 2 → Galaxy)
30-55 min  HÓA + ICH — Distill 2 items + Forced Link Rule (≥ 2 links)
55-70 min  NHỚ — Update CLAUDE.md OR targeted AI session
70-80 min  HÀNH — Harvest insights → Projects
80-90 min  Drill + Reflection (rotate Drills hàng tuần)
```

## Survival Conditions (dừng lại nếu vi phạm)

```
dJ/dt > dD/dt     Judgment grows faster than AI delegation
dG/dt > threshold  Galaxy vượt activation threshold (~200-300 notes)
dC/dt < 1/tháng   CLAUDE.md được refresh kịp với thực tế
```
