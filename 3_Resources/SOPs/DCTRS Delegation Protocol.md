# DCTRS Delegation Protocol — Workshop X AI Boss Framework

> Source: `D:\UxVina\workshop-x-delegation\CLAUDE.md`
> Dung khi delegate task cho AI trong Workshop X engineering context.

## 5 Elements — Moi Delegation PHAI Co Du

```
D — Delegate:   Task + Context + Output format
C — Constrain:  Allowed tools + Prohibited actions + Scope limits
T — Test:       Success criteria (how to verify output)
R — Review:     Which checkpoints need human approval
S — Specify:    Fallback on failure + Timeout + Version tracking
```

## Routine vs Ambiguous

| DELEGATE to AI (Routine) | DO YOURSELF (Ambiguous) |
|--------------------------|------------------------|
| Boilerplate code, config, docs | Architectural decisions, system design |
| Known algorithm implementation | Safety-critical logic validation |
| Code review pre-check | Vietnamese military context / TCVN |
| Tech spec formatting | Hardware debugging (thermal, EMI) |
