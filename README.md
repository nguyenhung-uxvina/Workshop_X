## Core Methodology

### THỊNH — Daily Operating Rhythm

| Step | Vietnamese | Action |
|------|-----------|--------|
| **T** | Thu (Capture) | Capture items into Inbox with "why" context |
| **H** | Hóa (Atomize) | Transform raw captures → atomic permanent notes |
| **I** | Ích (Connect) | Create ≥ 2 wikilinks per new note |
| **N** | Nhớ (Persist) | Review system docs for drift |
| **H** | Hành (Harvest) | Use Galaxy insights to inform real decisions |

### COD — Task Classification

- **Core** — Do it yourself (builds judgment)
- **Offload** — Delegate to AI under supervision
- **Default** — Skip or automate

### Galaxy (Zettelkasten)

Flat structure, no subfolders. Each note = one atomic concept, written in the author's own words with ≥ 2 wikilinks. Six topic clusters covering KM fundamentals, network effects, judgment & agency, AI failure modes, systems archetypes, and knowledge lifecycle.
